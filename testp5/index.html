The backward motion illusion occurs when the speed of rotation of an object is such that, in the interval between each iteration, the object moves nearly into the position it last occupied. If it occupies the exact position it previoulsy occupied, the object appears not to move

But if the object doesn't exactly match the interval (or a multiple of it), then the object will seem to turn slowly forward or backward. When the object is a little slow in reaching its previous position, the rotation appears to be backward; when the it arrives a fraction early, the visual impression is that the object is turning forward.

			var x=0;
			function setup(){
				createCanvas(1200,600);
			}
			function draw(){
				background(255);
				var n = sin(x)*600;
				ellipse(n+600,100,50,50);
				x=x+0.01;
			}
