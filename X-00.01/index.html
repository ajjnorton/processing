Random uniform movement

Lets imagine a little single celled animal (Proto) in search of food. One strategy would be to search randomly in the hope of stumbling upon something tasty. This implementation uses the the random() function to generate a random value in the range zero to eight. Each of the eight values corresponds to a direction as defined by the points on a compass. For example 0=North, 1=North East, etc. For each iteration a random direction is chosen. At each iteration there is a probability 12.5% (1 in 8) that Proto will take a step in any given direction. The random numbers we get from from the random() function are uniform and are not truly random.

        function step() {
            choice = int(random(8));
            var z = 1;
            if (choice == 0) {
                // Move east
                x += z;
            } else if (choice == 1) {
                // Move west
                x -= z;
            } else if (choice == 2) {
                // Move north
                y += z;
            } else if (choice == 3) {
                // Move south
                y -= z;
            } else if (choice == 4) {
                // Move north east
                y += z;
                x += z;
            } else if (choice == 5) {
                // Move south west
                y -= z;
                x -= z;
            } else if (choice == 6) {
                // Move south east
                y -= z;
                x += z;
            } else if (choice == 7) {
                // Move north west
                y += z;
                x -= z;
            }
            //x = constrain(x,400,width);
            y = constrain(y, -200, height);
            stroke(0, 0, 0, 50);
            ellipse(x + 400, y + height / 2, 10, 10);
        }
