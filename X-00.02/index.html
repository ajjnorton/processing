Backwards motion illusion

The backward motion illusion occurs when the speed of rotation of an object is such that, in the interval between each iteration, the object moves nearly into the position it last occupied. If it occupies the exact position it previously occupied, the object appears not to move.
But if the object doesn't exactly match the interval (or a multiple of it), then the object will seem to turn slowly forward or backward. When the object is a little slow in reaching its previous position, the rotation appears to be backward; when the it arrives a fraction early, the visual impression is that the object is turning forward. The code below creates a single ellipse that orbits with constant acceleration.

            var x=0;
            var y=0;
            var o=0;

            function setup(){
                var myCanvas = createCanvas(windowWidth, windowHeight);
                myCanvas.parent('can');
            }

            function draw(){
                //background(255);
                var n = sin(x)*100;
                var c = cos(y)*100;
                stroke(0);
                ellipse(n+(windowWidth/2),c+200,50,50);
                x=x+0.05+o;
                y=y+0.05+o;
                o=o+0.001;
                stroke(255);
                fill(0);
                //text(o,100,100);
                fill(255);

            }

            function windowResized() {
              resizeCanvas(windowWidth, windowHeight);
